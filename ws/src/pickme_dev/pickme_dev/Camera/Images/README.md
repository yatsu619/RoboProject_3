# RO45RobotikImages

Dieses Repository enthält Bilddaten für das RO45-Robotikprojekt, unterteilt in Produktiv- und Testdaten.

## Ordnerstruktur

```
RO45RobotikImages/
├── Productive/
│   ├── Cat/
│   ├── Circle/
│   ├── MagicUnicorn/
│   └── Square/
│   
├── Test/
│   ├── Cat/
│   ├── Circle/
│   ├── MagicUnicorn/
│   ├── Mixed/
│   └── Square/
├── LICENSE
└── README.md
```

## Beschreibung

### Productive/
Enthält die **Trainingsdaten**, gegliedert nach Kategorien:

- **Cat/** – Trainingsdaten der Kategorie Katze
- **Circle/** – Trainingsdaten der Kategorie Kreis
- **MagicUnicorn/** – Trainingsdaten der Kategorie Magische Einhorn
- **Square/** – Trainingsdaten der Kategorie Quadrat

### Test/
Enthält die **Testdaten**, gegliedert nach Kategorien:

- **Cat/** – Testdaten der Kategorie Katze
- **Circle/** – Testdaten der Kategorie Kreis
- **MagicUnicorn/** – Testdaten der Kategorie Magische Einhorn
- **Mixed/** – Gemischte Testdaten (mehrere Kategorien kombiniert)
- **Square/** – Testdaten der Kategorie Quadrat
